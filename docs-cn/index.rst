=================================
 Celery - Distributed Task Queue
=================================

Celery is a simple, flexible and reliable distributed system to
process vast amounts of messages, while providing operations with
the tools required to maintain such a system.
Celery是一个简单，可扩展和可靠的处理大量消息的分布式系统，同是提供了维护系统所必要的操作工具。

It's a task queue with focus on real-time processing, while also
supporting task scheduling.
他是面向实时处理的任务队列，同时提供任务调度。

Celery has a large and diverse community of users and contributors,
you should come join us :ref:`on IRC <irc-channel>`
or :ref:`our mailing-list <mailing-list>`.
Celery拥有大量和不同的社区用户和贡献者，你应该加入我们 :ref:`on IRC <irc-channel>` 或者 :ref:`our mailing-list <mailing-list>`。

Celery is Open Source and licensed under the `BSD License`_.
Celery是一个开源的，在`BSD License`_ 许可下。
